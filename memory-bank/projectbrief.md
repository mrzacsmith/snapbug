# Project brief

## Name

**SnapBug** (repository folder: `screenshot2-tool`)

## Purpose

A browser extension for capturing screenshots, preparing them for bug reports (annotation and upload described in manifest; implementation is evolving).

## Core goals

- Fast capture of the visible tab as PNG.
- Pass captured image into an annotation workflow.
- Support bug-report workflows (annotate, upload) as the product matures.

## Scope

- Primary deliverable: Chrome extension (Manifest V3) under `extension/`.
- Root `README.md` is minimal; detailed product narrative lives here and in `productContext.md`.

## Non-goals (current)

- Not specified; update as the product direction firms up.
